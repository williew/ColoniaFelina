# 🐾 Colonia Felina Hurlingham

Página web estática para promocionar la adopción responsable de la Colonia Felina Hurlingham, Buenos Aires.

## ¿Qué hacemos?

Somos un grupo de voluntarios que aplica el método **TNVR** *(Trap, Neuter, Vaccinate, Return — Atrapar, Esterilizar, Vacunar, Retornar)* para controlar de forma humanitaria la población de gatos callejeros en Hurlingham. Además los alimentamos diariamente y buscamos hogares para quienes puedan adaptarse a la vida doméstica.

## La página incluye

- **Hero** con foto strip real de los gatos y contadores animados
- **Sección TNVR** con explicación bilingüe del método y la marca universal del corte de oreja
- **Galería** con filtros por categoría (comiendo, tomando sol, descansando, gatitos, captura)
- **Sección de adopción** con guía de socialización para gatos semisalvajes
- **Contacto** con link al formulario de Google Forms

## Cómo correr el sitio

No requiere build ni dependencias. Dos opciones:

**Opción 1 — Abrir directo:**
```
Doble click en index.html
```

**Opción 2 — Servidor local:**
```bash
python -m http.server 7373
# Abrir http://localhost:7373
```
